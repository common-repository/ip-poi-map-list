The easiest plugin to use when your website needs to display Points Of Interest on a map.

== Description ==
POI Map List is the easiest plugin for displaying Points of Interest on Map.
All the Points of Interest are displayed on a customizable list and are linked to markers on map.
Users can filter the list by searching the specific content.

To display points of interest on your site, use our shortcode:
> <strong>`[POI_Map_list]`</strong><br>

== Features include: ==

* Search filter bar (name, address).
* Item details: image, phone, fax, email, url, description and address.
* Default picture set by the user
* Custom map (height).
* Custom marker and info window (color).
* Font and style based on your website theme
* Info window content details:title, address, phone, fax, email, url.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `ip-poi_map_list.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates

== Frequently Asked Questions ==

= What do I need to display the plugin on my site ? =

You only need a valid Google API Key.

= How about the customization ? =

The POI Map List allows you to customize the color of the markers and the information windows of the items on the map.
